什么是思维And如何思考

发表时间：2021-10-10 阅读： 评论：

现象的表面是一个节点，而思维是一个链条。

  • 向上可以挖掘事物的根本原因
  • 向下可以推断事物的发展结果

# 分析案例

#

# 向上挖掘

向上挖掘原因是为了找到解决问题最优解

  • 在挖掘事物根本原因时，若发现答案对解决问题没有意义就应该停止探讨（深度）

仅从限电政策出台来看，探讨疫情对生产力的影响似乎并不是解决问题的主线

  • 探讨的方向应该往可控的方向推进，避免不可控的因素（广度）

在上述案例中，电作为二次能源价格受一次能源供求关系影响，对于国家而言可控因素有：电价、电量、产品出口量，不可控因素有：自然资源、国外疫情、外贸订单量、进口原料价格。而在可控因素中电价是民生，为下选；而电量间接影响产品出口量，为上选。

# 向下挖掘

向下挖掘原因是为了预测结果和预料发展趋势，以便做出相应措施

  • 当某一级的概率低到对预案或者措施没有实际指导作用就可以停止了（深度）
  • 当现状发展到一些所预测级别时，后面所预测的概率就会自动增加了（结合实际）

# 系统性思考

不同视角看待问题会有不同结果，避免视角单一，需要全面、动态、深入思考。

  • 关注事实和数据
    • 列出数据陈述事实
  • 凭情绪和情感判断
    • 征求大家意见
    • 观察感觉上的变化
  • 充满希望正面思考
    • 提出思考方案
    • 陈述最终方案
  • 谨慎小心保守行事
    • 识别风险，沟通风险应对
  • 标新立异天马行空
    • 大胆寻求新的设想
  • 把控整个思考过程
    • 定义和描述问题
    • 针对问题发问
    • 概括和总结

# 动态思考

从线性思考到闭环思考，因果并非线性，该因之果可能是他果的因

例子：
员工效率低下导致项目延期，需要员工加班，员工产生抵触情绪，进而效率更低，项目延期更久

# 深入思考

从表面事件到系统结构，思考个别事件反映到系统的问题

例子：
一个员工离职可能是员工问题，多个员工离职就需要考虑是否是管理结构和公司氛围的问题

# 深入思考

从本位局限到关照全局、

例子：
你所在部门调整了工作流程，对接的部门也需要随之调整工作流程，平行部门也会妒忌

# 结构性思考

从事物的结构层面出发，来引导思维和组织表达的思维模式

  • 明确思考主题：思考有焦点，结论要先行，做事有目的
  • 归纳分组：空间框架（逻辑顺序）+ 时间框架（时间阶段）
  • 逻辑递进：主题 -> 方案 -> 执行细节

# 结论

思维决定出路，人和人的差距很可能就是思维上的差距
